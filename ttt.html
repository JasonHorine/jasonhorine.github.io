This is from a JavaScript homework assignment just two weeks into General Assembly's Web Immersive program. My personal goal was to build a game that would run very well on a tired old iPhone 3 we have sitting around the house.

Except for three lines of CSS and adding these comments, the code is as I wrote it then. I had plans for more functionality at the time, which you can see in the code. Getting stuck on a CSS issue prevented me from completing it before our class jumped into Ruby. Some features:

-Players take turns starting the game.

-Once complete, clicking the board anywhere clears it.

-The 'Your Turn!' arrow intuitively facilitates two players sitting side by side.
