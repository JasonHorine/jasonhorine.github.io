Tic Tac Toe

This game began as my deliverable for a JavaScript weekend homework assignment just two weeks into General Assembly's Web Immersive program. My personal goal was to build a game to run on smart phones. I especially wanted it to run well on old hand-me-down hardware, such as kids are most likely to have in hand.

Recent changes aimed to make the game a little more engaging for little ones by adding audible feedback. These changes also aimed to make the rules more readily apparent to newcomers.

    Future Plans:

  • Attach an X or O image (of the correct color) to the pointer dependent on which players' turn it is and the status of the cell the pointer is over.
  • Either have the 'winner' message show the player's marker and/or mark the winning row on the game board.
